Usefull Links

  • W3C.org

    W3C(World Wide Web Consortium), ist ein Internationaler Verbund von Standards für das World Wide Web (www), gegründet im Oktober 1994 von Tim Berner Lee. Das W3C arbeitet daran das Internet für alle zugänglich zu machen und einen Standard für dieses zu schaffen..

  • Html

    HTML ( HyperText Markup Language) ist eine textbasierte Auszeichnungssprache, die es uns erlaubt Webseiten zu erstellen, sie besteht aus Inhalt und Syntax, so wird eine HTML Datei Strukturiert, aber nicht formatiert. Die aktuellste Version ist HTML 5 und wurde vom W3C entwickelt. Desweiteren können in HTML Dokumenten weitere Informationen enthalten sein die auf der Webseite nicht direkt angezeigt werden, wie z.b. die Metainformationen die Angaben wie z.b. den Autor oder eine Beschreibung enthalten können.

  • CSS

    CSS, oder auch Cascading Style Sheets, ist eine Computersprache/ Gestaltungssprache, die vor allem zusammen mit HTML verwendet wird. CSS gibt dem Benutzer die Möglichkeit genau zu definieren wie Elemente der Webseite angezeigt werden soll. CSS wurde vom W3C entwickelt, der Begriff Cascading leitet sich aus dem Fakt ab das mehrere style sheets auf die selben Webseite angewendet werden kann.

  • semantisch korrrektes HTML

    Semantisch korrrektes HTML, steht dafür das der Inhalt einer HTML Semantisch strukturiert ist. z.b. der <p> tag zeigt an das der nachstehende text ein Paragraph ist, das hat für die Webseite Optische und Semantische Bedeutung Menschen wissen was ein Paragraph ist und Browser wissen wie sie diese darstellen sollen. In HTML4 gibt es veraltete tags wie <b> und <i> die den Inhalt nicht Semantisch strukturieren, sondern nur definieren wie der Inhalt angezeigt werden soll(Fett und Kursiv).

  • JavaScript

    JavaScript (kurz JS) ist eine Skriptsprache. Am häufigsten wird JS als teil von Browsern verwendet um z.b. Einfluss auf die Webseite zu haben(DOM), Client-seitig den angezeigte Inhalt ändern (z.b. Diashow), Dialogfenster angezeigt und genutzt werden, Formularangaben überprüft werden(noch vor dem senden). JS kann aber auch missbraucht werden um dem Webseitenbenutzer schaden zuzufügen, daher bieten viele neue Browser die Funktion an JS auf webseiten zu deaktivieren.

  • Validierung

    Validierung, steht für die Überprüfung von z.b. HTML Dateien diese werden nicht nur auf Fehler sondern auch auf Syntaktische Korrektheit überprüft. Nach der Validierung werden bem Benutzer mögliche Ursachen für für Fehler und abweichung von Standards angezeigt, sodass eine Überarbeitung vereinfacht wird. Das W3C bietet Validatoren für Webseiten relevante Dateien an.

  • Webstandards

    Webstandards sind Vorgaben für Webseiten, die ermöglichen sollen das Webseiten, fehlerfrei auf allen Browsern angezeigt werden können und für jeden zugänglich sind. Das W3C gibt Webstandards für HTML und CSS heraus, die sogenannten W3C-Recommendations, wie der Name schon sagt sind die Standards keien Pflicht sondern empfehlungen.

  • Accessibility

    Accessibility steht für die zugänglichkeit des Internets für Menschen mit Behinderungen. Wenn Webseiten korrekt gestaltet werden sind diese für jeden Menschen gleichmäßig zugänglich. So sind Bilder in einem Semantisch strukturierten HTML Dokument auch für Blinde Menschen Informativ, da der Inhalt des "alt" eines bildes diesem über Audio oder auch blindenschrift ausgegeben wird. Genauso sollten Texte und Bilder auch vergrößerbar sein damit diese auch für Menschen die schlechter sehen können erkennbar sind. Links sollten einfach anklickbar sein, damit auch menschen die evtl. keine maus benutzen diese anklicken können.

  • Usability

    Usability(oder auch Benutzerfreundlichkeit), steht für die Gestaltung von Webseiten, um diese möglichst Optimal für den Benutzer zu erstellen. So sollten Webseiten so gestaltet sein das die Texte einfach zu Lesen sind, Hintergründer nicht ablenken, um den Benutzer nicht zusätzlich zu beanspruchen. Genauso kann eine Webseite so gestaltet sein das die Struktur interessant ist und die Aufmerksamkeit des Lesers aufrecht erhält.

  • XML

    XML, steht für Extensible Markup Language. XML ist eine Auszeichnungssprache die zur Strukturierung und beschreibung von Daten dient, die sowohl von Menschen als auch von Maschinen gelesen werden kann. Das W3C definierte die XML 1.0 Standards. XML wurde ursprünglich dazu entwickelt um Inhalt und Formatierung zu trennen.
